# Instagram Clone (Mobile App)

A mobile application inspired by Instagram, built using **React Native** for the frontend and **Node.js** for the backend. The app allows users to create accounts, share photos, like and comment on posts.

## Tech Stack

### Frontend
- React Native
- Expo
- React Navigation
- Axios

### Backend
- Node.js
- Express.js
- PostgreSQL / MongoDB
- JWT Authentication

## Features

- User authentication (Sign up & Login)
- User profiles
- Upload and share photos
- Like and comment on posts
- View posts feed
- RESTful API integration
